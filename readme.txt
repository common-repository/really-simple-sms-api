== Installation ==

= The easy way: =

1. Go to the Plugins Menu in WordPress
1. Search for "Really Simple SMS API"
1. Click 'Install'
1. Activate the plugin

= Manual Installation =

1. Download the plugin file from this page and unzip the contents
1. Upload the `really-simple-sms-api` folder to the `/wp-content/plugins/` directory
1. Activate the `really-simple-sms-api` plugin through the 'Plugins' menu in WordPress
